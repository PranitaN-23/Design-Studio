# Design-Studio
A modern and elegant **Interior Design Website** created for a client who is a professional interior designer.   This project was developed with the help of AI tools to assist in design structure and animation, while the layout, styling, and customization was handled using **HTML** and **CSS**.

## Features
- Home, About, Services, Gallery, and Contact sections  
- Fully responsive design for desktop and mobile  
- Smooth navigation and visually balanced layout  
- Professional color palette and typography  
 
## Technologies Used
- **HTML5**
- **CSS3**
- **AI-assisted design** (for structure and layout suggestions)
